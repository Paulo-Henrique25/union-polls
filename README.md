# Union Polls

Aplicação de enquetes em tempo real desenvolvida como desafio frontend da Union Developers.

## ✨ Funcionalidades

- Criar enquetes
- Adicionar múltiplas opções
- Sistema de votação em tempo real
- Atualização automática com Supabase Realtime
- Status da enquete:
  - Em andamento
  - Não iniciado
  - Finalizado
- Responsividade mobile
- Validação de formulários
- Testes unitários com Vitest

## ⚙️ Como rodar localmente

Clone o projeto:

```bash
git clone https://github.com/Paulo-Henrique25/union-polls.git
```

Entre na pasta:

```bash
cd union-polls
```

Instale as dependências:

```bash
npm install
```

Crie um arquivo `.env.local`:

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
```

Inicie o projeto:

```bash
npm run dev
```
